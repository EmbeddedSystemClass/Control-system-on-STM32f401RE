@par Directory contents

 - project .ewd/.eww/.ewp: A pre-configured project file with the provided library 
                           structure that produces an executable image with IAR 
                           Embedded Workbench.
                
 - include : This folder include .h files with discription of memory and main functions.

 - src: This folder include description of init, math, fuction, settings...


For use it you need STDPERIPH_DRIVER, from ST-miroelectronix site. Board with STM32F401RE microcontroller and programmer.
